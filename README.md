# 노씨라이트-이열
##프로젝트 소개 
이 프로젝트는 시각장애인이 지팡이의 진동으로 지하철 문을 찾는 웹 시뮬레이터입니다.

## 이동약자 유형
-시각장애인

## 주요 장면
1. 출발 지점(지하철역)
2. 사람 장애물
3. 도착 지점(지하철 문)
4. 결과 화면

## 조작 방법
- W: 앞으로 이동
- A: 왼쪽 이동
- S: 뒤로 이동
- D: 오른쪽 이동
- 마우스 클릭으로 시점 이동, 한 번 더 클릭 시 시선 이동 멈춤

## 체험 포인트
- 지팡이가 지하철 문 위치로 잘 안내하는가?
- 사람들과 부딪히면 어떻게 되는가?
